

# iOS开源框架使用指南

### Not Only Demo简介

Not Only Demo 以开放平台sdk和主流的APP开发框架为基础搭建一套可以帮助开发者快速开发APP的脚手架.在此基础上开发者可以一天开发一款智能硬件控制APP.


### Not Only Demo下载地址

下载地址 https://github.com/C-Life/clife_open_ios_sdk 。
